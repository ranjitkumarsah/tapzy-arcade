# Screenshots for the pitch

Drop PNG/JPG captures here with these **exact filenames** so they show up in
[`../PITCH.md`](../PITCH.md):

| File | What to capture |
|---|---|
| `launcher.png` | Home screen — the game grid with coin chip + streak/daily buttons |
| `game.png` | A game mid-play (Bubble Shooter or 2048 look great) |
| `daily.png` | The Daily Spin wheel (mid-spin or on the reward with confetti) |
| `wallet.png` | Wallet modal — withdrawable vs bonus balance + "Watch ad to earn" |
| `store.png` | Store modal — the Telegram Stars products |
| `leaderboard.png` | Leaderboard with the medal rows |
| `demo.gif` *(optional, best)* | 15–30s screen recording: launcher → play → win with confetti |

## How to capture (best quality)

- **Phone (recommended):** open the Mini App in Telegram, use your phone's screenshot,
  then crop to the app area. Use both a light and a dark example if you like.
- **Desktop:** Telegram Desktop → open the Mini App → screenshot the window.
- **Screen recording → GIF:** record a short clip on your phone, convert to GIF (e.g.
  ezgif.com) and save as `demo.gif`. A short animated demo is the single most
  persuasive asset for selling the code.

Aim for ~1080px wide, well-lit, no personal chat visible in the frame.
